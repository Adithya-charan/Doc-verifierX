# DocVerifierX

An AI-powered document verification system that uses OCR and NLP to automatically extract, analyze, and validate data from uploaded documents.

## Features
- OCR-based document scanning
- Intelligent text extraction and structuring
- Automated data verification
- Flask + React-based web app

## Tech Stack
- Frontend: React.js
- Backend: Flask (Python)
- AI/ML: Tesseract OCR, NLP, and Deep Learning

## Setup
1. Clone the repository  
2. Install dependencies  
3. Run the frontend and backend  
4. Deploy using Vercel (frontend) and Render (backend)
